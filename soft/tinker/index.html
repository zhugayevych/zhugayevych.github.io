Installation: Windows

  • Unzip the downloaded "Complete Distribution" to a desired program folder, I use C:/Sci/tinker.
  • Unzip "Executables" to, let say, C:/Sci/tinker/bin subfolder.
  • Edit and copy tinker.bat to the program folder (or any other location).
  • Associate tin-files with tinker.bat. For this purpose you can edit and use these file associations.
  • Optionally create task,task1,task2,... folders where you will run TINKER jobs (one per physical core).
  • Sample input files: tin, inp, key (modify path to force field parameter files).

Installation: Linux

  • PBS submission script: primitive, advanced. To submit a job run qsub tinker/_pbs -N test1
  • Keep force field parameter files in a common directory, e.g. ~/lammps/_prm.
