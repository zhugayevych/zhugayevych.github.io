Installation

  • Install to a nonsystem folder with simple path, e.g. C:/Sci/PSPad.
  • Copy Syntax files to the program Syntax folder.
  • Install dictionaries to Spell folder.
  • Run PSPad as administrator.
    • Uncheck Program (part 2)/Show Splash Screen
    • Uncheck System integration/Multi User Environment
    • Check System integration/Register in System context menu
    PSPad.ini file will be created in the program folder. Otherwise that file will be kept here.
  • If PSPad does not appear in system context menu run this registry patch.
  • Make a copy of PSPad.ini file from program folder (the original file cannot be edited in PSPad). Selectively update that copy by the local PSPad.ini file. Save the new file locally and put it instead of the existing PSPad.ini in the program folder.
  • Edit and run FileAssoc.reg.
  • Run EditHTML.reg to set PSPad as default HTML-editor.
  • Rename pdb.dll from /Plugins/Filetype/ if you want to treat pdb-files as text files, otherwise PSPad saves them in unreadable format.

Notes

  • Installation of other software and system updates may corrupt PSPad installation so that it must be reinstalled. Examples of such software: POV-Ray.
  • To avoid opening files in HEX mode set AutoDetectBinary=0.
  • Remember to set RealTabs appropriately.
