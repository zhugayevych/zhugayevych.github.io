Installation

  • I would recommend to add no version number to the installation directory.
  • Edit eps2pdf.bat and use it to convert eps files to pdf. Copy it to C:/Sys or any other folder in PATH. Use FileMenuTools for right-click access. Avoid confusion with same-name Ghostscript and MiKTeX commands, e.g. gs/lib/ps2pdf.bat and miktex/miktex/bin/x64/ps2pdf.exe.
  • The same is for myps2pdf.bat and pdf2txt.bat.
