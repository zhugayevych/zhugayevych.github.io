Installation: Windows

  • If you have an existing installation just copy it.
  • For a new user obtain the license to unpack the package.
  • Edit and copy firefly.bat to the program folder (or any other location).
  • Associate fly-files with firefly.bat. For this purpose you can edit and use these file associations.
