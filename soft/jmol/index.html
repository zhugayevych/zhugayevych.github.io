Installation

  • Make sure that Java is installed.
  • Unzip the downloaded package to a desired program folder, I use C:/Sci/Jmol.
  • Unzip macroses to the user .jmol/ folder (if needed create this folder in your user root folder, e.g. C:/Users/azh/.jmol, by mkdir command).
  • Edit and copy Jmol.bat, JmolCube.bat, Cube.spt to the program folder (or any other location).
  • Associate jmol-files with Jmol.bat. For this purpose you can edit and use these file associations.
  • To create a link in Windows Programs folder set Target to "C:\Program Files\Java\jre7\bin\java.exe" -Duser.language=en -Xmx1024m -jar Jmol.jar -L -t -i
  • There are two editable configuration files in .jmol: history and properties
