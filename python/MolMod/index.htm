Description of MolMod package
(only 5 routines out of 100 are implemented in their simplified versions)

by Andriy Zhugayevych, updated 13.09.2016

MolMod is a molecular modelling package.

If you have comments, suggestions, or have found a bug, please email me.

See also this package for Maple.

Installation

  • Download MolMod.zip. Change the default settings in MolMod.py (one block only).
  • The documentation is organized as follows: if you need help on, say, ReadXYZ and WriteXYZ routines open ReadXYZ.ipynb.
  • My system is Anaconda on Windows 7.
