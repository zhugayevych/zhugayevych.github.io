Description of BasicTools package
(only 2 routines out of 100 are implemented in their simplified versions)

by Andriy Zhugayevych, updated 13.09.2016

BasicTools package provides miscellaneous routines lacking in standard Python distributions.

If you have comments, suggestions, or have found a bug, please email me.

See also this package for Maple.

Installation

  • Download BasicTools.zip.
  • The documentation is organized as follows: if you need help on, say, ReadBIN and WriteBIN routines open ReadBIN.ipynb.
  • My system is Anaconda on Windows 7.
