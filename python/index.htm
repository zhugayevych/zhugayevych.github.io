Python/IPython: bugs, tools, applications

Python installation instructions

Python/IPython bugs and problems from a user perspective (2014)

Note: BasicTools package is required for all the applications listed below.

Packages:

  • BasicTools – miscellaneous routines lacking in standard Python distributions
  • SSHTools – supports SSH network protocol
  • MolMod – molecular and crystal modeling
  • Siman by Dmitry Aksenov – supports calculations in VASP and Abinit

Scripts:

  • GeometryOptimization – sample script for geometry optimization in Gaussian on a remote machine (to test it you may need this file)
  • SinglePoint – sample script for single point calculations in Gaussian on a remote machine
