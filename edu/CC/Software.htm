Required sofware

Important: The software provided within the course activities can be used only for your personal education within the scope of the course

Contact person: Sergey Matveev, Andriy Zhugayevych

  • Any computing cluster account:
    • for education use Skoltech Virtual Machine (firstletteroffirstname.lastname@10.17.11.165, unzip setupVM to your $HOME folder)
    • for research use Skoltech cluster (firstname.lastname@pardus.skoltech.ru, ask helpdesk)
    • for CEE CREI projects you can use CEE cluster
    • as last resort use course-dedicated server (ask Sergey Matveev)
  • Any program for
    • visualizing isosurfaces in molecules and crystals: Jmol, Vesta
    • classical molecular mechanics: LAMMPS (available on cluster, install also on your computer)
    • quantum chemistry of molecules: MOPAC (academic), Gaussian (commercial, available on cluster)
    • quantum chemistry of crystals: VASP (commercial, available on cluster)
  • Optionally but highly recommended:
    • any molecular builders and graphical interface to solvers: Avogadro, Ascalaph, BALLview, Spartan (Skoltech has educational license),
    • OR any fast-programming language for pre- and post-processing: IPython (see packages and notebooks here), Maple with MolMod package (Skoltech has educational license)
  • Additional tools which may be needed: Open Babel (converts chemical formats), p4vasp (VASP visualization tool)

See also here and CEE CREI computational facilities

Notes on using the above software

  • If you do not have IPython but have account on CEE mini-server you can use our IPython server for calculations (test files)
  • Attend this ISP course if needed: Computing Life with Linux
  • Open Babel: go to Control Panel -> Region and Language -> Additional Settings and set Decimal symbol to "."
