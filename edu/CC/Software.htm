Required sofware

Important: The software provided within the course activities can be used only for your personal education within the scope of the course

Contact person: Artem Naumov, Sergey Matveev

  • Any computing cluster account:
    • Skoltech cluster (firstname.lastname@pardus.skoltech.ru)
    • CEE CREI cluster
    • course-dedicated server
  • Any program for
    • visualizing isosurfaces in molecules and crystals: Jmol, Vesta
    • classical molecular mechanics: LAMMPS
    • quantum chemistry of molecules: MOPAC (academic), Gaussian (commercial), Molpro (commercial)
    • quantum chemistry of crystals: VASP (commercial)
  • Optionally but highly recommended:
    • any molecular builders and graphical interface to solvers: Avogadro, Ascalaph, BALLview, Spartan (Skoltech has educational license),
    • OR any fast-programming language for pre- and post-processing: IPython, Maple with MolMod package (Skoltech has educational license)
  • Additional tools which may be needed: Open Babel (converts chemical formats)
