  1. In Windows Regional and Language Options install English and Ukrainian languages with one keyboard layout per language.
  2. In Language Bar select switching combination, e.g. Ctrl+Shift, and hide the bar.
  3. In Keyboard Layout Manager create or import your own layouts, e.g. ukru.klm2000. Exit KLM and reboot.
