Setup

Activate Windows. If fails, validate Windows first. Then use Microsoft Genuine Advantage Diagnostics Tool if needed. Next run WAT Fix. Finally use Windows Loader. Local versions of these tools are stored here.

Control Panel

  • Autoplay: Mixed Content – open folder.
  • Date and Time: enable automatic synchronization.
  • Default Programs: set your favorite file associations by manually patching Windows Registry, see my code here.
  • Desktop Gadgets: see the table below.
  • Device Manager: check for malfunctioning devices.
  • Devices and Printers: install printers including print-to-file printers.
  • Display: native resolution, 125% font size.
  • Indexing Options: check.
  • Internet Options: see Internet Explorer below.
  • Mouse: adjust speed, hide while typing, wheel scrolls 1 line.
  • Notification Area Icons: disable annoying icons and notifications.
  • Personalization: set Desktop background color to (43,85,128), run Appearance11.reg for medium size title and menu and Appearance10.reg for small size. See also help.
  • Power Options: power button hibernates, closing the lid turns into sleep; sleep after 15/30 min, hibernate after 30/120 min, turn off disk after 10/30 min, turn off display after 5/15 min, dim display after 2/5 min; display brightness 30%, dimmed 15%.
  • Program and Features: remove "garbage".
  • Regional and Language Options: adjust Format (choose Russian or Ukrainian to read Cyrillic letters in text files), set current Location, Ukrainian or Russian system locale (HKLM\System\CurrentControlSet\Control\Nls\CodePage\ACP will be set to 1251), change keyboard layouts by Keyboard Layout Manager or other programs.
  • System: name the computer, disable Remote Assistance; environment variables: add "C:\Sys" to PATH.
  • System/Visual Effects: Show thumbnails, Show window contents, Smooth edges, Smooth-scroll.
  • Taskbar and Start Menu: lock, auto-hide, use small icons, never combine; customize Start Menu; see the table below.
  • Windows Update: automatically check for updates but let me choose whether to download them.

  • Add my Network Locations to Computer folder.

Startup and resident programs and services

  1. Remove unwanted programs from HKLM,HKCU\Software\(Wow6432Node\)Microsoft\Windows\CurrentVersion\Run and other instances like HKU\S-1-5-21-3385893834-3931371485-2020306079-1001\Software\Microsoft\Windows\CurrentVersion\Run.
  2. Configure Startup by disabling unnecessary programs using System Configurations utility.
  3. Disable other resident programs by Services utility.

Tips and tricks

  • Configure cmd-window: run cmd, choose Defaults, Font: Lucida Console 14, Layout: 120×1000, 120×60
  • To prevent a cmd-window from closing use cmd /k "command" (the command must be enclosed in "").
  • start /affinity 0xD command will run the command on CPU3, CPU2, CPU0 because 0xD=1101.
  • Icons see here: Install folder, IconArchive.
  • To clean-up Explorer Navigation panel change HKCR/CLSID/{key}/ShellFolder/Attributes as follows (the 1st value is to enable, the 2nd is to disable):
      Homegroup {B4FB3F98-C1EA-428d-A78A-D1F5659CBA93} b084010c b094010c
      Libraries {031E4825-7B94-4dc3-B131-E946B44C8DD5} b080010d b090010d
      Network   {F02C1A0D-BE21-4350-88B0-7367FC96EF3C} b0040064 b0940064
  • Use SFC utility to check system files.
  • To block web-parasites update your HOSTS file by MVPS list. For efficient browsing stop "DNS Client" service from automatic loading.
  • Set up a beep when Caps Lock is pressed.
  • To change background color of Windows Photo Viewer to black run WinPhotoViewerColors.reg.
  • To free up some space reduce the shadow storage maxsize by vssadmin command.
  • If you have problem with viewing hidden files change HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced\Folder\Hidden\SHOWALL\CheckedValue from 0 to 1.
  • To delete index.dat files see here.
  • Use Windows Sysinternals (description) if needed.
  • For security issues see KLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies.
  • Any program must be run as administrator to configure it.
  • To remove Recycle Bin from desktop delete the key HKLM\Software\Microsoft\Windows\CurrentVersion\Explorer\Desktop\NameSpace\{645FF040-5081-101B-9F08-00AA002F954E}="Recycle Bin"
  • To restore Recycle Bin to desktop you can try this (not tested).
  • If "Devices and Printers" doesn't work, make sure that Bluetooth service is running.
  • To remove defective network printer empty PRINTERS spool and restart Print Spooler using services.msc.
  • Gadgets are not loading: run regsvr32 msxml3.dll as admin, then restart sidebar.exe
  • Autolock screen all users: run gpedit.msc, go to UserConfiguration/AdministrativeTemplates/ControlPanel/Personalization, enable ScreenSaver, PasswordProtect it, set ScreenSaverTimeout, ForceSpecificScreenSaver to %windir%\system32\rundll32.exe user32.dll,LockWorkStation, run gpupdate /force
  • Use net user to list network names for local accounts. Use \Domain_name\User_name to switch to other domain.
  • Use netsh winsock reset to clean network cash.
