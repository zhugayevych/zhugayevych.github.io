Setup

  • To disable opening pdf in browser disable the plugin from inside the browser.
  • Disable resource-eating Adobe services by cleaning Registry/Run, Task Scheduler, blocking them by msconfig.exe or services.msc, and renaming their running files:
    • AdobeARM.exe and armsvc.exe from here
    • AAM Updates Notifier.exe and updaterstartuputility.exe from here
    • CS6ServiceManager.exe from here
  • Check if you need
    • FlashUtil64_11_9_900_117_ActiveX.exe from here
    • FlashUtil32_11_9_900_117_ActiveX.exe from here
  • Disable automatic updates. Don't update it.
  • Preferences:
    • Page Display: Two-Up, Fit Page
    • Units: Millimeters
    • Updater: Do not download or install updates automatically
  • Upper toolbar:
    • Create (nonremovable)
    • Print Ctrl+P
    • Edit current tool set (nonremovable)
    • Set initial view
    • Save file as... Shift+Ctrl+S
    • Save as reduced file size PDF, as optimized PDF
    • manual separator
    • Add sticky note
    • Highlight/Insert text
    • Add: text, text box, bookmark
    • Edit Text and Images
    • manual separator
    • Delete, Extract, Insert, Crop page
  • Lower toolbar:
    • Page number
    • Previous/Next views Alt+Left/Right
    • Selection/Pan tools
    • Zoom to selected area
    • Zoom out/in Ctrl+Minus/Plus
    • Fit width and scroll
    • Fit one full page
    • Show two full pages
    • Rotate clockwise/counterclockwise Shift+Ctrl+Plus/Minus
    • Click and selct area for snapshot
    • Find text
  • Memorize other keyboard shortcuts:
    • Save Ctr+S
    • Hide/show menu bar F9
    • Hide/show toolbars F8
    • First/Last page Home/End
  • Create your own Tool Set

Tips

  • When optimizing PDF, internal and external links can be deleted by "Discard all comments, forms and multimedia".
  • If text direction is wrong try change Preference/Type/Text Engine.
