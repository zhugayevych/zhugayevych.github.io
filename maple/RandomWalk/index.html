Description of RandomWalk package

by Andriy Zhugayevych

RandomWalk is a collection of tools related to a random walk problem.

If you have comments, suggestions, or have found a bug, please, email me.

Tested for Maple 17 on Windows 7 and 10.

Installation

  • Download source files RandomWalk.zip, unzip to any location, open install.mw and follow the instructions.
  • Windows users: If failed to install or you don't need the source, download RandomWalk_lib.zip and unzip to Maple library folder MapleXX/lib.
  • Experienced users: the package can be initialized by reading RandomWalk.ini with Maple "read" command.
  • The documentation is built into the Maple Help system. Individual help pages are also available as individual mw-files.
