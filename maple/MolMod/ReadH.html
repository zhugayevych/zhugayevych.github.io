File tools: Hamiltonians

See examples in ReadH.mw

Procedures

  • ReadTBH(filename,/ids,{longid,printout})::SG,M,P,tbH,GV,GP,GI,HV,HP,HI read tight-binding Hamiltonian file, where
    • ids::{[Hid::{0,posint,string}],[Gid,Hid]}:=[0] if nonzero identifies parameter set (if only Hid is given, its prefix is used to determine Gid),
    • longid::boolean use long indexing [m1,m2,t1,..,td,o1,o2],
    • SG::string space group,
    • M::Matrix(nd,nt) unit cell matrix,
    • P::Matrix(nd,nm) site positions in lattice coordinates,
    • tbH::table Hamiltonian as described here,
    • GV,HV::list(string) Geometry and Hamiltonian symbolic variables,
    • GP,HP::table(id=list(name=value)) Geometry and Hamiltonian parameters,
    • GI,HI::list(string) indices of GP,HP ordered as in filename,
    Important: if symbolic variables are assigned before ReadTBH call, the result is unpredictable.
