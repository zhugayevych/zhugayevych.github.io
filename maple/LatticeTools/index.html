LatticeTools package

LatticeTools package provides routines for working with lattices (finite and infinite, regular and generic).

Download source or library or ini-file. Tested for Maple 2024.

Prerequisite packages: BasicTools

The package is under major revision, the working part of the package is available as LatticeTools.ini
