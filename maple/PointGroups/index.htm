Description of PointGroups package

by Andriy Zhugayevych

PointGroups is a package for manipulating point groups.

If you have comments, suggestions, or have found a bug, please email me.

Tested for Maple 17 on Windows 7 and 10.

Installation

  • Download source files PointGroups.ini and PointGroups.mw to any location, open PointGroups.mw and follow the instructions.
  • Windows users: If failed to install or you don't need the source, download PointGroups_lib.zip and unzip to Maple library folder MapleXX/lib.
  • Experienced users: the package can be initialized by reading PointGroups.ini with Maple "read" command.
  • The documentation is built into the Maple Help system. Individual help pages are also available as individual mw-files.
