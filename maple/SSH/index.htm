Description of SSH package

by Andriy Zhugayevych

SSH package supports SSH network protocol via PuTTY or similar client as well as job management via TORQUE or SLURM.

If you have comments, suggestions, or have found a bug, please email me.

My system is Maple 14 on Windows 7. Tested for Maple 17.

Installation

  • Download source files SSH.ini and SSH.mw to any location, open SSH.mw and follow the instructions.
  • Windows users: If failed to install or you don't need the source, download SSH_lib.zip and unzip to Maple library folder MapleXX/lib.
  • Experienced users: the package can be initialized by reading SSH.ini with Maple "read" command.
  • If not exist, create SSH_Setup.ini in UserConfigurationFolder (see its definition here). Use SSH_Setup.ini as a template.
  • The documentation is built into the Maple Help system. Individual help pages are also available as individual mw-files.
